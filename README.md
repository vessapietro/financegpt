# FinanceGPT

App financeiro pessoal com dashboard, histórico mensal e chat de lançamentos.

## Como usar

1. Suba estes arquivos para o repositório GitHub `financegpt`.
2. Na Vercel, conecte o repositório.
3. Configure as variáveis:
   - NEXT_PUBLIC_SUPABASE_URL
   - NEXT_PUBLIC_SUPABASE_ANON_KEY
4. Deploy.

## Recursos

- Dashboard financeiro
- Histórico por mês
- Chat financeiro
- Cancelar lançamento
- Salvamento local no navegador
- Estrutura pronta para Supabase
