# FinanceGPT

App financeiro pessoal com dashboard, histórico mensal e chat de lançamentos.

## Deploy

1. Apague todos os arquivos antigos do repositório.
2. Suba todos os arquivos desta pasta na raiz do GitHub.
3. Na Vercel, deixe Output Directory vazio.
4. Faça Redeploy.

## Variáveis Vercel

- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY

## Funcionalidades

- Dashboard financeiro
- Histórico mensal expansível
- Chat para registrar gastos e receitas
- Cancelar último gasto
- Cancelar lançamento pela tabela
- Salvamento local no navegador
- Layout dark premium
